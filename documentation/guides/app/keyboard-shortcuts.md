# Keyboard Shortcuts

The API Client supports keyboard shortcuts to speed up your workflow. On macOS, shortcuts use ⌘ (Cmd). On Windows and Linux, use Ctrl instead.

## Browser & Desktop App

| Shortcut | Action |
| --- | --- |
| ⌘ Enter | Send request |
| ⌘ K | Open command palette |
| ⌘ B | Toggle sidebar |
| ⌘ L | Focus address bar |
| Escape | Close modal |

## Desktop Only

These shortcuts are available in the desktop app.

### Tabs

| Shortcut | Action |
| --- | --- |
| ⌘ T | New tab |
| ⌘ 1–8 | Jump to tab 1–8 |
| ⌘ 9 | Jump to last tab |
| ⌘ ⌥ ← | Previous tab |
| ⌘ ⌥ → | Next tab |
