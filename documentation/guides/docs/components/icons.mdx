# Icons

Icons render scalable vector graphics from the built-in [Phosphor](https://phosphoricons.com/) library or from a direct URL.

## Icon Sources

`src` accepts two forms:

1. **A Phosphor key** in the format `phosphor/<variant>/<name>`. Variants are `regular`, `bold`, `duotone`, `fill`, `light`, and `thin`. Examples: `phosphor/regular/house`, `phosphor/fill/heart`, `phosphor/bold/check-circle`. See the [Phosphor icon list](https://phosphoricons.com/) for available names.
2. **A direct URL** to an SVG or image file. Useful for custom or brand-specific icons.

## Properties

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `src` | `string` | _required_ | Phosphor key (e.g. `phosphor/regular/star`) or full URL to an SVG/image. |
| `src-dark` | `string` | — | _MD only._ Dark-mode variant. Not available on `<Icon>` in MDX. |
| `title` | `string` | — | Accessible label read by screen readers. |
| `size` | `string` | — | CSS size for both width and height (e.g. `16px`, `1.5em`, `2rem`). |

### Light and Dark Variants (Markdown only)

`src-dark` lets you swap the icon source in dark mode. This prop is only available on the markdown form.

```md Markdown
<scalar-icon
  src="phosphor/regular/sun"
  src-dark="phosphor/fill/moon"
  title="Theme">
</scalar-icon>
```
