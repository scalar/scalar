# Scalar.Azure.Functions Documentation

Welcome to the documentation for the `Scalar.Azure.Functions` package. This guide shows how to render the
Scalar API reference from an Azure Functions app (isolated worker). For a comprehensive list of all available
configuration options, please refer to the [integration documentation](https://scalar.com/products/api-references/integrations/azure-functions/integration).
