# Scalar Icons

The icon library that we use to provide user selectable icons.

## Installation

```bash
npm add @scalar/icons
```

## Usage

```vue
<script setup lang="ts">
import { LibraryIcon } from '@scalar/icons/library'
</script>

<template>
  <LibraryIcon src="basic-shape-diamond" />
</template>
```
